# Object Detection

Object detection using classical methods (HOG + sliding window, template matching) and/or pre-trained deep learning models.

## Topics Covered
- Sliding window detection
- Template matching
- Background subtraction and motion detection
- Using pre-trained models (e.g., Haar cascades, YOLO via OpenCV DNN)

## Usage
```bash
python main.py --image <path_to_image>
```
