# Image Classification

Image classification using traditional machine learning (HOG + SVM) and/or deep learning.

## Topics Covered
- Feature extraction (HOG, LBP, color histograms)
- Classical classifiers (SVM, k-NN)
- Dataset loading and splitting
- Evaluation metrics (accuracy, confusion matrix)

## Usage
```bash
python main.py --dataset <path_to_dataset>
```
