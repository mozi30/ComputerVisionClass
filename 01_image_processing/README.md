# Image Processing

Basic image processing techniques using OpenCV and NumPy.

## Topics Covered
- Loading and displaying images
- Color space conversions (BGR, RGB, Grayscale, HSV)
- Filtering and smoothing (blur, Gaussian, median)
- Edge detection (Sobel, Canny)
- Morphological operations (erosion, dilation)
- Histograms and histogram equalization

## Usage
```bash
python main.py --image <path_to_image>
```
